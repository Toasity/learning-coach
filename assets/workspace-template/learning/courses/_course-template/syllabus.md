## 第一性原理问题探索

- 章节核心概念、理论及其关系：待盘点。
- 现实产业/公共系统/研究或工程领域：待识别。
- 锚点问题与决策者：待定义。
- 决策、预测、诊断或设计任务：待定义。
- 目标与成功标准：待定义。
- 可观测输入或证据：待定义。
- 边界：待定义。
- 硬约束、权衡与失败代价：待定义。
- 事实：待识别。
- 假设：待识别。
- 惯例：待识别。
- 当前课程层级的第一性基础：待识别。
- 核心概念到锚点问题的映射：待记录。
- 显微镜模型及其忽略的现实因素：待定义。
- 补充子问题：待定义。
- 已覆盖核心点：待记录。
- 未覆盖核心点：待记录。

锚点问题合格条件：存在有后果的决策；具有目标、证据、约束或权衡和失败代价；
通常需要整合至少三个章节核心概念；不能退化为单一定义或公式的代入题；移除产业
背景后，推理、约束或成功标准会发生实质变化。

推导链：问题 → 事实/假设/约束 → 第一性基础 → 推导 → 正式概念 → 解法 → 反事实测试。
